    ```go
    }

    type Options struct {
        Since string    // duration or ISO date

    }
    ```

    **ParseSince function** (exported, used by cmd/diff.go):
    ```go
    func ParseSince(s string, now time.Time) (time.Time, error)
    ```
    - Try ISO date formats FIRST (per pitfall 6): RFC3339, "2006-01-02T15:04:05", "2006-01-02"
    - Then try `duration.Parse(s)` which returns `time.Duration` directly (per pitfall 2)
    - Apply `now.Add(-dur)` -- NO `* time.Second` multiplication (cf's Parse returns Duration, not int)
    - If now.IsZero(), use time.Now()
    - On all failures: return error with message `invalid --since value %q: expected duration (e.g. 2h, 1d) or date (e.g. 2026-01-01)`


    - Accepts pre-fetched version data (the command handles API calls)
